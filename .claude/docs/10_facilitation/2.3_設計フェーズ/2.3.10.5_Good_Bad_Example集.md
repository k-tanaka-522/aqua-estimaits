# 2.3.9.5 Good_Bad_Example集（Mermaid図）

## 目的

設計フェーズのMermaid図のGood/Bad例を提供します。

---

## ❌ Bad Example: 技術詳細が不足

```mermaid
flowchart LR
    A[User] --> B[Server]
    B --> C[DB]
```

**問題点:**
- 具体的なAWSサービス名がない
- ネットワーク構成が不明

---

## ✅ Good Example: 技術詳細を明記

```mermaid
flowchart LR
    A[User] --> B[ALB]
    B --> C[ECS Fargate]
    C --> D[RDS PostgreSQL]

    style C fill:#ff9900
    style D fill:#c8e6c9
```

**改善点:**
- 具体的なAWSサービス名を明記
- 色分けで視認性向上
