# 2.6.5.1 デプロイ手順書 - 前提条件・準備セクション

## テンプレート

```markdown
# デプロイ手順書

## 前提条件

### 必要なツール
- AWS CLI 2.x以上
- Docker 24.x以上
- kubectl 1.28以上（EKS使用時）

### 必要な権限
- AWSアカウント管理者権限
- またはデプロイに必要な最小権限セット

### 必要な情報
- AWSアカウントID
- デプロイ先リージョン
- 環境名（staging/production）

## 事前準備

### 1. シークレット情報の準備
- AWS Secrets Managerにシークレット登録済み
- データベース接続情報の確認

### 2. ドメイン・SSL証明書の準備
- Route 53にドメイン登録済み
- ACMでSSL証明書発行済み

### 3. バックアップの取得
- 既存環境のバックアップ取得（本番環境の場合）
```
